# Partnered Content
Third-party content officially published on D&amp;D Beyond implemented for the Aurora Character Builder for D&amp;D 5e

Partnered Content Index URL: https://raw.githubusercontent.com/FreshRoyal/elements-partnered/master/partnered.index

This repo also contains homebrew featured through official Wizards of the Coast and D&amp;D Beyond channels, along with homebrew released by D&D design team members.

Featured Homebrew Index URL: https://raw.githubusercontent.com/FreshRoyal/elements-partnered/master/featured-homebrew.index

This repo will also eventually include content from the first six books of Goodman Games' Original Adventures Reincarnated series.


## Disclaimer
The Partnered Content repository is intended as a place for easily-accessible Aurora Builder content (xml) files of products you already own. Please ensure you only access content in accordance with your local laws.

All images are property of the respected owners and are stored for static links.
